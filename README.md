# lexi
#### Small command line Interactive English Dictionary (Lexicon)


##### Installation:

```sh
pip install --user git+https://github.com/paalso/lexi.git
```

##### Usage:
```sh
lexi word
```
or
```sh
lexi "words collocation"
```
